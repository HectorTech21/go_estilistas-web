Política de Privacidad

En Go Estilistas nos comprometemos a proteger la privacidad de los usuarios.

Responsable del tratamiento

Titular: Go Estilistas
Email: info@goestilistas.com

Datos que recopilamos

Podemos recopilar datos personales como:

  • Nombre
  • Teléfono
  • Correo electrónico

Estos datos se recogen a través de formularios de contacto o comunicación directa (por ejemplo, WhatsApp).

Finalidad del tratamiento

Los datos se utilizan para:

  • Gestionar consultas o solicitudes
  • Contactar con el usuario
  • Ofrecer información sobre nuestros servicios

Legitimación

La base legal para el tratamiento de datos es el consentimiento del usuario.

Conservación de los datos

Los datos se conservarán el tiempo necesario para cumplir con la finalidad para la que fueron recogidos.

Derechos del usuario

El usuario puede ejercer sus derechos de: acceso, rectificación, supresión y oposición. Enviando una solicitud al correo electrónico indicado.

Seguridad de los datos

Go Estilistas adopta las medidas necesarias para garantizar la seguridad de los datos personales.

Cambios en la política

Nos reservamos el derecho a modificar la presente política para adaptarla a novedades legales.
